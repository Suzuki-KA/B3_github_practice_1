# README.md

## プロジェクト名

**Simple Python Greeting(挨拶プログラム)**


## 概要

このプロジェクトは，**Pythonの入出力機能**を使って，ユーザーから名前を受け取って，それに基づいたメッセージを表示するアプリケーションです．

## 特徴

- コードが単純で動作確認をすぐにできる．
- 入力によって出力される文章が変化する．


## 動作環境

- **OS:** Pythonが動作する環境であれば何でも可
- **Python:** 3.12.3

## インストール方法

1. **hello.py**という名前でファイルに保存する．

## 使い方

ターミナル（コマンドプロンプト）を開き,**greeting.py**を保存したディレクトリに移動して,以下のコマンドを実行する．

```bash
python greeting.py
